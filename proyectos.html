Proyectos y herramientas

Ejemplos de automatización, reporting y análisis de datos desarrollados en el área Home Merchandising.

Informe semanal de ventas Hogar

Consolidación SAS/SQL de datos de ventas y stock de 50 divisiones. Reporte automático validado para Dirección en menos de 24 horas.

Tecnologías: SAS, SQL, Excel, Power BI

Sistema de alertas de surtidos

Herramienta de detección automática de errores de surtido y codificación. Reduce roturas y mejora la calidad del dato.

Tecnologías: SAS, SQL, Python

Ranking de roturas y éxito de venta

Informe automatizado de roturas, cobertura y rendimiento de referencias, con alertas semanales a los responsables de departamento.

Tecnologías: SAS, Excel VBA

Base de datos de consulta Hogar

Repositorio centralizado de referencias, stock y ventas actualizadas semanalmente. Permite análisis homogéneos entre divisiones.

Tecnologías: SQL, SAS, Excel

Informe de promociones y elasticidades

Modelo analítico para evaluar el impacto de campañas promocionales sobre ventas y margen. Soporte a decisiones comerciales.

Tecnologías: R, SAS, Excel
